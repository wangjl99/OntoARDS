
POI_disease = named clinical disease, syndrome, or disorder with established diagnostic criteria.
  Examples: ARDS, COVID-19-associated ARDS, Hyperinflammatory ARDS subphenotype, Pneumonia, Sepsis
  NOT: outcomes (mortality), mechanisms (NF-kB activation), contexts (ICU setting), phenotypes without disease status

POI_disease vs POI_phenotype boundary:
  Use POI_disease when the concept has ICD/SNOMED disease code
  Use POI_mechanism when it describes a pathophysiological process
  Use POI_context when it describes a clinical circumstance or setting
  Use POI_outcome when it describes a measured endpoint
